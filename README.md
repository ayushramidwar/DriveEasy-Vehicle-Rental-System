# DriveEasy Vehicle Rental System

## Overview

DriveEasy is a Vehicle Rental Management System developed using Java, Spring Boot, MySQL, and REST APIs.

## Features

* User Registration
* User Login
* Vehicle Management (CRUD)
* Booking Management (CRUD)
* MySQL Database Integration
* RESTful APIs
* API Testing with Postman

## Tech Stack

* Java 17
* Spring Boot
* Spring Data JPA
* MySQL
* Maven
* Postman
* Git & GitHub

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer
* Entity Layer
